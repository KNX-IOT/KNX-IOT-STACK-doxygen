◆ py_discover_devices_with_query()

kisCS_EXPORT void py_discover_devices_with_query ( int  scope,
const char *  query 
)

discover KNX devices on the network function can be used for discovery with serial number:

  • ?ep=urn:knx:sn.[serial-number] :device with specific serial-number
  • ?if=urn:knx:ia.[Individual Address] :device with specific Individual Address
  • ?if=urn:knx:if.pm :devices in programming mode
  • ?if=urn:knx:if.o : devices with specific interface (e.g. if.o)
  • ?d=urn:knx:g.s.[ga] : devices belong to a specific group address
Parameters
scopethe scope (2 or 5 site local)
querythe query
